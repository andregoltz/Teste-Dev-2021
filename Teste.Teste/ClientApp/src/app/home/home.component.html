Descrição do Teste

Desenvolver uma aplicação que possua uma interface de login a qual deverá validar se o usuário e senha informados existem no banco de dados antes de permitir o acesso a tela principal.

Após o login, a tela principal deve apresentar um menu simples com um item chamado Cadastro de Clientes.

O menu Cadastro de Clientes deverá encaminhar para uma nova página, a qual deve possuir uma listagem de todos os clientes cadastrados e um botão para inserir um novo cliente. A listagem de clientes deve ter a funcionalidade de permitir a edição do cliente selecionado. Essa edição deve ser feita em uma nova página, carregando todos os dados do cliente e permitindo alterar as informações. O botão novo cliente deve direcionar para a mesma página utilizada para edição, porém trazendo os campos em branco para que seja possível incluir um novo cliente.

A listagem deve apresentar os seguintes dados do cliente:

  • Nome
  • CPF
  • Data de nascimento

A edição deve apresentar os mesmos campos da listagem e mais uma lista de telefones (DDD) e Número para que seja possível incluir e deletar diversos contatos para o cliente. Antes de incluir um cliente devemos validar se já não existe um cliente com mesmo CPF. Antes de incluir um telefone devemos validar se o telefone já não existe para aquele cliente.
